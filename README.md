Marketing site for mosibello.com (sanity).

## Installing

```bash
# From Source
git clone https://github.com/jeremyd123-din/jd-consulting.git
cd jd-consulting
```

then install dependencies

```bash
  npm install
```

## Extra options

```bash
# Run a production build, locally
npm run dev

# Serve a production build, locally
npm run build
npm start
```
